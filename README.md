## 注意事项
- 部分手机需要开启应用的弹窗权限才能显示开发者菜单：设置->应用管理->(应用名称)->权限管理->悬浮窗
- 不要在`render`中使用`this.setState()`
- `props`、`state`设置后不会马上变化，需要等到`render`后
